void copyStringWithLength ( char *  dest,
const char *  src,
size_t  destSize 
)

Copy the first destSize characters of the string pointed to by src, including the terminating null byte ('\0'), to the buffer pointed to by dest.

Warning
The strings may not overlap, and the destination string dest must be large enough to receive the copy.
If src is longer than destSize characters, the string is truncated and the terminating null byte ('\0') is added.
Parameters
destthe destination string
srcthe source string
destSizethe maximal number of characters to copy
